Instructions for Using the Title V State Access to Care Assessment Tool

1. Registration

To fill out and submit the survey, you need to have a username (a valid email address that will be used to contact you) and password. All the members of each group that will be filling out the survey together—for example, everyone on the Minnesota team will work together to fill out one survey; everyone on the Mississippi team will fill out another survey together—will share an email address and password. Note: while everyone on your team can all fill out and submit the same survey, only one person at a time can safely edit the survey. If two or more people are editing the same survey at the same time on different machines, someone's entries are likely to be overwritten when someone else does a save, and one or more of you will lose your work. So make sure to take turns.

To register, go to the login page—http://syssci.renci.org/SAT/index.html—and send an email to the address shown. In your email include your name and affiliation.

2. Logging in

The site adminstrator will register your team's email address and notify you of your password. Then you can log in: go to http://syssci.renci.org/SAT, enter your email address and password into the appropriate spaces, and click the "Submit" button.

IMPORTANT: Use Chrome or Firefox to log in to the State Assessment Tool. Internet Explorer and Safari may not work.

3. Home page

When you've logged in successfully you'll be looking at a page that shows a link labeled "Log out," another link that says "Help with downloaded surveys," a list (initially empty) of all the surveys that have been saved by everyone on your team (that is, everyone who logs in to the SAT using the same email address and password that you use), and, at the bottom, a button labeled "Create New Survey."

3.1. Log out

Click on this link to close the survey and return to the login page. For full security make sure you've logged out of all instances of the survey.

3.2. Help with downloaded surveys

This is a link to a page that explains the layout of downloaded files.

3.3. The List of Saved Surveys

When you first log in to a new account this list will be empty. Each survey that you save will be added to this list. The list is ordered by date and time modified, most recent first. There are four columns for each saved survey.

3.3.1. Survey ID

The first column shows the numerical name assigned to each survey. The system assigns this name at the time a survey is saved. To open a saved survey, in order to look at it or edit it, click on its Survey ID.

3.3.2. Name

The second column shows an optional name that you may have chosen to the survey in addition to the numerical name (in column 1 automatically assigned by the software. See section 3.3.7 below to learn how to create this name.

3.3.3. "Created At"

The third column shows the date and time of creation for the survey in Greenwich Mean Time (GMT). To get the local time find out how many hours difference there is between GMT and your own time zone, and subtract that difference. So, for instance, if your geographical location lies in the Eastern Time Zone and it's Daylight Saving Time, then the time where you are is four hours behind GMT, so subtract four hours from the "Created At" value to get your local time of creation.

3.3.4. "Modified At"

Column four shows the date and time (in GMT) a survey was most recently saved.

3.3.5. "Download"

Click on "download" in the fifth column to download the survey for that row. The downloaded file can be imported into Excel or equivalent program in CSV (Comma-Separated Values) format. See this page for details of the file format.

3.3.6. "Owner Email""

This column displays the email address of the account used to create the survey described for that row.

3.3.7. "Rename"

Click on "rename" in column seven to create or modify the name for that survey as displayed in column two.

3.3.8. "Delete"

Click on the "X" in the eighth column for a survey you wish to remove entirely from the database. Warning! Once it's deleted, it's gone forever, so be sure this is what you want to do.

3.4. Creating a New Survey

At the bottom of the home page there's a button that's labeled "Create new survey": click it to create a new blank survey.

4. Always Available in the Survey Footer

4.1. The Save Button

Saves the survey you're filling out to the server database just as it is. It receives a number which you can use to retrieve it again later, to look at it or modify it.

4.2. Instructions

Opens (in a new tab or window) this Instructions page that you're looking at right now.

4.3. Navigation Tool

It looks like this:

<prev Contents Overview Instructions 0 1 2 3 4 5 Download next>

Use this to move around between the various sections of the survey. Click on "<prev to move to the previous page (you won't go anywhere if you're on the first page—"Contents"—which is where you will be when you first open or create a survey). Click on "next>" to move to the next page (if you're already on the last page—"Download"—you won't go anywhere). Click on anything in between those two and you'll go to the corresponding page. The page you're currently on is highlighted in red in the Navigation Tool.

4.4. Single page

Click here to view the whole survey as a single unbroken page. You can always go back to the multi-page format by clicking on the Navigation Tool.

4.5. Log out

This logs you out of the survey.

5. Printing as a PDF File

You should be able to print out the survey from any mainstream web browser. Each browser will have its own approach, but as a representative example: In Chrome 44 on the Mac (OS X 10.10.4), select “Print”, then “Open PDF in Preview" and then “Save.” There is a “Save as PDF” option for printing in Windows 8 Chrome version 39, and probably other versions as well.

In other cases (e.g., Firefox and Internet Explorer on Windows), see this article which says “you’ll need a print-to-PDF program that interfaces with Windows as a print driver. There are plenty available, and you might have one installed that you don’t even know about. In researching this article, I discovered that I have four. If you don’t have a PDF option among your print drivers, download and install the free version of the BullZip PDF Printer.”

When you're printing, you may want to make sure you're in single page mode, so that the whole survey gets printed, rather than just the currently selected section.

6. Getting Back to the Home page

You can return to the home page from anywhere in the survey by clicking on the colorful National MCH Workforce Development Center logo in the left-hand corner at the top of each section.

7. Importing a Downloaded Survey .csv File into Excel

The ".csv" extension on downloaded survey files stands for "Comma-Separated Values." Excel and other spreadsheet programs offer mechanisms for importing .csv files so that their contents are distributed across a number of separate cells in a spreadsheet for easier reading and manipulation. Simply opening a .csv file using the File Open dialog may work just fine, in which case the rest of these instructions may be ignored. Sometimes, however, using "File Open" may cause the entire contents of a .csv file to be treated as a single unpartitioned block of text so that the whole thing is dropped into a single cell. In such an eventuality it will be visible but it probably won't be usefully formatted.

Note that different versions of Excel and its ilk have different ways of reading in .csv files. These instructions describe the .csv import process for Excel version 14.5.9. They may not apply to your version or to other spreadsheet programs. If not, please consult the documentation that accompanies your software or search online.

  • In Excel, select "New Workbook" from the "File" menu.
  • Select "Import..." from the "File" menu.
  • A dialog box titled "Import" will appear. Make sure the "CSV file" radio button is selected, and click the "Import" button.
  • Using the standard dialog that appears (it's called "Choose a File" in this version), navigate to the folder on your hard drive that contains the .csv file you wish to import. For file type (the dropdown menu here is called "Enable:") choose "CSV Files." If that's not an option with your spreadsheet program, or if the file you want to import is not visible in the dialog, try "All Files" or another file type. Double-click on the file you're interested in or select it (single-click) and click on the "Get Data" button.
  • The next dialog box that appears is titled "Text Import Wizard - Step 1 of 3." Make sure the "Delimited" radio button is selected and that you're starting import at row 1. Click "Next."
  • In the next dialog, titled ""Text Import Wizard - Step 2 of 3," make sure under "Delimiters" that only the "Comma" radio button is checked (in Excel 14.5.9 "Tab" always needs to be explicitly deselected). "Treat consecutive delimiters as one" should not be checked. Click "Finish."
  • The next dialog is called "Import Data." It asks "Where do you want to put the data?" Select the radio button labeled "Existing sheet:" and use the default position =$A$1 (the top left corner of the workbook). Click "OK."
  • You'll probably need to set the formatting in cells B4 and B5 (to the right of the cells that read "Created At" and "Modified At") to be Numbers of the Date category. Choose a Type that shows time in addition to day, month, and year.
  • See Help with downloaded surveys for more information on what's in the file you just imported. That same information is also available from a link with the same title on the survey's home page.
