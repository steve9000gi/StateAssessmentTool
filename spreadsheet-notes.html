State Title V Roles in Health Reforms including the Affordable Care Act:
A Title V State Access to Care Assessment Tool

Help with downloaded surveys

When you download a survey, you should be able to load it into any spreadsheet software, including Microsoft Excel and Google Sheets. Once you do, here are some notes about how the data is displayed.

  1. The first 10 rows are metadata about the survey and have a substantially different format than the actual survey data. The general approach is that column A contains the field, and column B contains the value. The exceptions are rows 9 and 10, which contain the 5 names and 5 affiliations, respectively, in columns B-F.
  2. Rows 11-15 contain the followup text from the bottom of each section.
  3. Everything after row 16 is the survey data. The general format is:
    • Column A: section number (e.g. "s1")
    • Column B: question number (e.g. "q1" or "q2b")
    • Column C: question type (see below)
    • Column D: text from "list" areas (e.g. Section 1, Question 5, where it says, "If yes, please list specific partners:")
    • Column E: text from "notes" areas
    • Columns F-: depends on the question type
  4. Question types:
    selectOne
    Only one answer is possible. The selected answer is in Column F. The first answer is 1, the second answer is 2, etc. If no answer was selected, the value will be "_". In a few cases (e.g. Section 2, Question 4), the survey allows text to be entered as an explanation for choosing an answer labeled "other". In such cases, the "other" text is in Column G.
    selectMany
    Multiple answers are possible, as in, "check all that apply". Columns F on contain a 1 if the corresponding answer was checked, and a 0 otherwise.
    selectOneThenSelectMany
    Some questions, like Section 5 Question 1a, are fundamentally a "selectOne" question, but there are additional checkboxes under one of the answers. As with the "selectOne" question type, Column F contains the primary answer and Column G contains any "other" text. The following columns contain the sub-answers as in the "selectMany" question type. Note that sub-answers are only possible if the first primary answer is selected, so all of the values after Column G will be 0 unless Column F is 1.
    CLOA
    This is a "Capacity and Level of Activity" question, like in Section 1, Question 7. Column F is the Capacity from 0-5, and Column G is the Level of Activity from 0-5, which may be blank if the user did not select a Level of Activity.
  5. The text from "notes" and "list" areas can contain multiple lines. If they do, the lines will show up in a single spreadsheet cell with \n separating them.
